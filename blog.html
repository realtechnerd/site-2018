Adding a Favicon to your website

Monday, May 7, 2018 | 7:37 AM | Coding Tutorial

Hello! Welcome to Coding Tutorials #1! Today, I will show you how to add a Favicon to your website. First, go to the <head> section. Under the <title>, paste this piece of code in there. <link rel="favicon" type="image" href="#">. Now, all you need to do, is fill out the name of the logo (E.g. mylogo.png, mylogo.jpg, etc.), in the hashtag. If it doesn't work, please email me at help@realtechnerd.com.

Samsung Chromebook 3

Monday, May 7, 2018 | 7:02 AM | Tech Reviews | ☆☆☆

Hello! Today, I'm going to review the Samsung Chromebook 3. The Samsung Chromebook 3, is a Chromebook by Samsung and Google. I'm amazed with the sleekness of this Chromebook. The Samsung Chromebook 3, was launched on Feburary 2017. The Chromebook has a Intel HD Graphics 400, and it has a built-in 720p camera. The Chromebook also has Bluetooth 4.0, 802.11b/g/n/ac. It has a micro-SD slot and a HDMI slot as well. This has 2 USB ports, and one headphone/speaker jack. You can choose to pick RAM as well, and I actually picked 4 GB of RAM. It also has a TPM (Trusted Platform Module), a small chip inside of your computer, made by Intel, for cyber-security. You get a 1-year limited warranty with the device. The CPU or Processor is a Intel Celeron N3060 / 1.6 GHz. The laptop comes with 32 GB of SSD - (eMMC). This laptop overall, is okay. I actually use this laptop for everyday use, and I think it's okay. It does lag sometimes, when you are running a lot of tabs on Chrome. It's okay though. I wouldn't really recommend this laptop, but you should atleast try it. Even though I am a Samsung Fan, I will rate this about 3 stars.
You can buy the Samsung Chromebook from Google Shopping.

Code Pad IDE

Sunday, May 6, 2018 | 8:18 PM | Tech Reviews | ☆☆☆☆☆

Hello! Today, I wanted to show this IDE called Code Pad IDE. I used this code editor to make my website. Code Pad IDE, is a IDE (Intergrated Development Environment), that was made by Andrew Borg, a developer from Malta. This was initially released on December, 27, 2018. Code Pad IDE can support more than 50 programming languages, including Python and C++. Code Pad IDE is really good in working with HTML5. I found the design and simplicity of this website, immaculate. This IDE is kind of like Virtual Studio, looking at more of the design perspective. This IDE is compatible, with pretty much any Operating System that supports the Chrome Browser. When I think about it, I think this is actually one of the best IDEs out there, way ahead of Notepad++ and Eclipse. I would definitely rate this 5 stars because of its simplicity.
You can install Code Pad IDE from the Chrome Web Store.
